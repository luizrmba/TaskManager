# TaskManager

Esse repositório foi criado para a disciplina de **Controle de Versão de Software**.

## 📌 Sobre o Projeto
O **TaskManager** é um projeto desenvolvido para praticar conceitos de versionamento, organização de código e fluxo de trabalho com Git e GitHub.  
O objetivo é criar um sistema simples de gerenciamento de tarefas, permitindo cadastrar, listar, atualizar e remover tarefas.

## 🚀 Tecnologias Utilizadas
- Linguagem: Python  
- Ferramentas:
  - Git  
  - GitHub

## ✅ Funcionalidades
- Criar uma nova tarefa  
- Listar tarefas existentes  
- Marcar tarefa como concluída  
- Editar descrição de tarefa  
- Excluir tarefa  

## 🛠 Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/luizrmba/TaskManager.git && cd TaskManager
2. Crie um ambiente virtual (opcional):
   ```bash

   .venv\Scripts\activate

3. Rode o .py
   ```bash
   python TaskManager.py
